== Description ==
Tired of testing website search solutions that never end indexing or way too expensive to implement?
Whats about do it in a single click?

[Searchbar.org](https://searchbar.org) is the easiest way to implement a search widget to your website, it works in any device with a responsive layout and predictive results.

== Installation ==
1. Create your [Searchbar.org](https://searchbar.org) account.
2. Type your website address in the user page.
3. Hit Test & Install button. A new tab will open with a demo page
4. Copy the integration script and paste it in your Searchbar.org Wordpress plugin admin page.
5. Click "Save Changes".

Now you can go to any page in your website and you will see an amazing search plugin.

== Frequently Asked Questions ==
= How do i customize my widget layout? =
The basic settings are available in any paid plan, like change colours and positioning.
We also offer api access for custom layouting & apps/datasets integration in our professional and corporate paid plan.

= Can I connect multiple domains in a single search bar? Ex: landing pages, Wordpress blog, stores and support. =
Yes in any plan with api access.

= Searchbar can be connected to a dataset? =
Yes in any plan with api access.

= Can I hide a page from the results? =
Yes is just set your page to not be indexed in google. We also give filter options on paid plans, that allows you to set URL masks.

Is Searchbar.org completely free?
Yes, our services are free until your website have reached 50 indexed pages, after that you search will continue working but it may display ads in your search results feed or you can upgrade to a plan that fits your site size.
